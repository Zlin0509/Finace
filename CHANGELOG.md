# Changelog

## 0.3.1 - 2026-07-16

### Verified

- Go 全量测试、随机顺序测试、竞态检测、`go vet` 和格式检查通过。
- Python `0.2.x` 兼容层 35 项测试通过，前端 JavaScript 语法检查通过。
- 隔离 SQLite 迁移、完整性、`0600` 权限、备份、导出和设置脱敏通过。
- 真实 510300 净值、缓存一致性、回测和 Walk-Forward 结果通过。
- 桌面 1440px、移动 390px 与最窄 320px 浏览器流程通过，控制台无错误。

### Changed

- 新增 `0.3.1` 全量测试报告和可复现发布校验记录。
- Go 服务与嵌入式前端版本提升到 `0.3.1`。

## 0.3.0 - 2026-07-16

### Added

- Go 原生 HTTP 服务、SQLite 持久化、基金行情客户端和嵌入式 HTML/CSS/JS 前端。
- 二次元欢迎首页与本地 WebP 背景，桌面侧边栏和移动端底部导航。
- 买入持有、双均线、动量回测及 Walk-Forward 参数搜索。
- 参数稳定性、正收益窗口、跑赢基准比例、过拟合差距与 A-D 可靠性评级。
- Go 单元测试、HTTP 嵌入资源测试和量化性能基准。

### Changed

- Go 单二进制成为默认产品运行路径，Python/Streamlit 标记为 `0.2.x` 兼容实现。
- 基金净值优先从单次历史文件解析，并使用 6 小时进程内缓存。
- Walk-Forward 主指标改为串联所有样本外日收益后统一计算，回撤可跨窗口累计。
- 移动端将导航固定到底部，新增交易作为独立第 5 项动作。

### Fixed

- 拒绝负数或过小的训练/验证窗口，避免外部请求触发切片越界。
- 优化器不再原地重排调用方的价格序列。
- 移动端累计成本换行、环图图例裁切和浮动按钮遮挡问题。

### Performance

- Apple M3 Pro 上完整 10 年回测约 `55.8 us/op`，指标路径约 `33.2 us/op`，快速 Walk-Forward 约 `0.354 ms/op`。

## 0.2.0 - 2026-07-16

### Added

- SQLite 本地数据库，持久化交易流水、模型配置和数据源配置。
- 旧 `data/portfolio.json` 自动迁移及迁移前备份。
- 每日自动备份、设置页手动备份和无密钥持仓 JSON 导出。
- `fund storage-info`、`fund backup`、`fund export-data` 和 `fund ui` 命令。
- 可随 wheel 安装的默认配置资源。

### Changed

- 持仓从交易流水实时重建，避免派生汇总与原始交易不一致。
- 卖出交易校验可用份额。
- 项目版本提升到 `0.2.0`。

### Security

- 本地数据库、WAL 文件与备份在 POSIX 系统上限制为 `0600`。
- 持仓导出不包含本地保存的 API Key。
- SQLite 内容未额外加密，完整数据库备份仍需按密钥文件保管。
